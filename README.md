# Titanic Data Analysis - Week 2

## Overview
This notebook explores relationships in the Titanic dataset using correlation analysis, scatter plots, and categorical comparisons.

## Steps
- Cleaned dataset (handled nulls, duplicates, data types)
- Generated correlation heatmap for numeric columns
- Visualized strongest correlations with scatter plots
- Explored categorical relationship (survival by gender)

## Key Findings
1. **Passenger Class vs Fare** – First-class passengers paid much higher fares than third-class.  
2. **Family Size (SibSp vs Parch)** – Families with siblings often also had parents/children onboard.  
3. **Survival by Gender** – Females had nearly double the survival rate compared to males.
